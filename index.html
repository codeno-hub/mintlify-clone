Getting Started / Quickstart

Quickstart

Set up and deploy your documentation site in under 5 minutes. No configuration required.

Write in Markdown

Author docs in familiar Markdown or MDX. Full React component support for interactive content.

Git-based workflow

Connect your GitHub repo. Every push auto-deploys your docs with zero configuration.

AI-powered search

Intelligent search understands context. Users find answers instantly with natural language queries.

OpenAPI Support

Generate beautiful API reference from your OpenAPI spec. Interactive playground included.

1

Install the CLI

bash 
$ npm i -g mintlify
2

Start the dev server

bash 
$ mintlify dev
Tip: Requires Node.js 18 or higher. Run node -v to check your version.
